# Premier League Insights (POO + EDA + ML)

Proyecto académico en Python para analizar estadísticas de jugadores de la Premier League
utilizando Programación Orientada a Objetos, Análisis Exploratorio de Datos y visualización.

## Estructura del proyecto
- `src/`: código fuente (clases, servicios, visualización)
- `data/raw`: CSV original (`premier.csv`)
- `data/processed`: CSV limpio/enriquecido
- `notebooks/`: notebooks de trabajo (`01_EDA.ipynb`, `02_Visualizacion.ipynb`)
- `dashboard/`: app opcional de Streamlit (`app.py`)

## Requisitos sugeridos
Python 3.10+
pandas, matplotlib, seaborn, plotly, scikit-learn, streamlit, jupyter

## Ejecución rápida
1. (Opcional) Crear entorno virtual
2. Instalar dependencias
3. Colocar `data/raw/premier.csv`
4. Ejecutar notebooks o `streamlit run dashboard/app.py`
